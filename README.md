# FurFinder — Gestión de Usuarios


## Menú interactivo

Al ejecutar `npm run dev` se abre el menú en consola:

```
────────────────────────────────────────
  FURFINDER — Gestión de Usuarios
────────────────────────────────────────
  1. Crear usuario
  2. Listar usuarios
  3. Buscar por id
  4. Actualizar usuario
  5. Desactivar usuario
  0. Salir
```

---

## Cómo ejecutar

```bash
npm install
npm run dev
```

> Requiere Node 18+ y TypeScript 5+. El runner utilizado es `tsx`.
